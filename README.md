# Gudangku API 🚀

REST API untuk sistem manajemen gudang berbasis Laravel.

## 🔧 Tech Stack
- Laravel 12
- Laravel Sanctum (Authentication)
- MySQL
- Filament (Admin Panel)

## ✨ Fitur Utama
- 🔐 Authentication (Login & Register)
- 📦 Manajemen Produk
- 🏭 Manajemen Gudang
- 📊 Inventory Tracking
- 🔄 Stock In / Stock Out / Transfer
- 🚨 Low Stock Detection

## 📡 Endpoint Utama
- POST /api/v1/login
- POST /api/v1/register
- GET /api/v1/products
- POST /api/v1/stock-in
- POST /api/v1/stock-out
- GET /api/v1/inventories
